<h1>Happy</h1>
 Happy é um projeto web que visa facilitar visitas aos orfanatos próximos à você.
 

<h2>Tecnologias</h2>:
* Node.js
* Express
* SQLite
* handlebars


<h2>Sobre</h2>
Este projeto foi desenvolvido durante a **NLW#3**, realizada pela **_@Rocketseat_** do dia 12 a 18 de Outubro de 2020.
